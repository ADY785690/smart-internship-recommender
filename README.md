# 🎯 AI Career & Internship Intelligence Platform

An AI-powered internship and career recommendation system built using Python, Pandas and Streamlit.

## Features

- Internship Recommendation
- Career Match Analysis
- Resume Score Prediction
- Expected Salary Estimation
- Skill Gap Detection
- Company Recommendations
- Personalized Career Roadmap
- Learning Resource Suggestions
- Downloadable Career Report

## Technologies

- Python
- Pandas
- Streamlit

## Career Paths Supported

- Data Analyst
- ML Engineer
- Software Engineer
- Quant Analyst

## Live Demo

Deploy using Streamlit Cloud.
