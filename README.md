# 🎯 AI Career & Internship Intelligence Platform

An AI-powered platform that helps students discover the best internships, identify skill gaps, estimate salary potential and evaluate placement readiness.

---

## 🚀 Features

### Internship Recommendation
- Data Analyst
- ML Engineer
- Software Engineer
- Quant Analyst

### Career Match Analysis
- Role suitability scoring
- Career comparison dashboard
- Personalized recommendations

### Placement Readiness Score
- Readiness score out of 100
- Performance evaluation

### Skill Gap Analysis
- Missing skill detection
- Improvement suggestions

### Salary Prediction
- Estimated salary range
- Career growth insights

### Personalized Learning Roadmap
- Weekly action plan
- Career preparation guidance

---

## 📊 Dashboard Metrics

- Career Match
- Placement Readiness
- Salary Prediction
- Skill Gap Count
